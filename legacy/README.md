# Legacy — Arquivos Originais

Esta pasta contém os arquivos da **versão inicial** do projeto, antes da refatoração para o Nintendo Stock Quest.

| Arquivo | Descrição |
|---|---|
| `teste.html` | HTML original — Dashboard de Metas com tema Mario |
| `teste.css` | CSS da versão de teste (variante experimental com dupla borda) |
| `template.php` | Fragmento PHP com o componente de barra de progresso e blocos |

## Por que foram preservados?

Servem de referência histórica para entender a evolução do projeto:
- `teste.html` → `frontend/index.html` (refatoração completa)
- `teste.css` → `frontend/style.css` (reescrita com tema NES fiel)
- `template.php` → descontinuado (substituído pelo frontend estático + backend FastAPI)
